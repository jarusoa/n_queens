1. In order to excute the program type "make"  to compile the program
and then type "make run" to run the program.

IN THE PROGRAM
1. Simply type an integer number for the "N" the program will run with, and press enter
2. After that the program will produce the queen positions and the chess board for a 
graphical view to see how it looks.


x. You can type "make run" to do another run of the program or "make clean"
to clean up the compliled files required to run the program.
